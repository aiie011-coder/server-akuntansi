Periode Lock
Kunci periode bulanan agar jurnal tidak bisa diedit/dihapus. Terkunci otomatis saat Buka Periode Baru — atau kunci manual per bulan.
Klik Refresh untuk memuat daftar periode.
Audit Trail
Klik Refresh untuk memuat log audit.
Buka Periode Baru
Cara kerja: Fitur ini akan mengambil saldo akhir periode berjalan dari setiap akun, lalu menetapkannya sebagai saldo_awal periode baru. Semua jurnal, penyesuaian, dan data transaksi periode berjalan tidak dihapus — tetap tersimpan di perusahaan/periode lama.
Nama perusahaan/buku untuk periode baru
Periode lama akan dikunci otomatis saat periode baru dibuat — semua bulan yang memiliki jurnal akan terkunci agar tidak bisa diedit. Anda tetap bisa membuka kunci manual dari tabel Periode Lock jika diperlukan. Periode berjalan tidak dihapus.

🔄 Sinkronisasi Data Multi-User
Cara Kerja Sinkronisasi
Aplikasi ini bekerja offline — data tersimpan di masing-masing komputer.
Untuk berbagi data antar pengguna, gunakan file backup .zeqsync.
1️⃣
Sebelum mulai kerja
Minta file .zeqsync terbaru dari rekan kerja → klik Merge Data → data Anda akan diperbarui
2️⃣
Setelah selesai kerja
Klik Export Backup → kirim file .zeqsync ke rekan kerja via WhatsApp, email, atau Google Drive
⚠️
Penting diingat
• Jika dua orang edit data yang sama persis, versi terbaru yang menang
• Jika salah satu menghapus data, penghapusan akan diikuti sisi lain
• Data baru dari keduanya selalu digabung otomatis
